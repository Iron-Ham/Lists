# ``ListKit``

A high-performance diffable data source and snapshot engine for UICollectionView.

## Overview

ListKit provides the low-level primitives that power the Lists framework: a custom diffable data source, flat and hierarchical snapshots, and a fast O(n) Heckel diff algorithm. Use ListKit directly when you need full control over collection view data sources or want to integrate with your own abstraction layer.
